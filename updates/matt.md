Matt Pocock's Skills v1.1 system formalizes Agentic Software Engineering by replacing unpredictable prompting with a rigid, markdown-based "harness" that forces AI agents to follow deterministic engineering procedures, such as /wayfinder for project navigation and /implement for TDD-based coding [1]. The architecture centers on organizing repositories to use local skills (e.g., grill-with-docs, to-tickets) that break work into atomic, verifiable tasks, mitigating context drift [1]. Key skills include /wayfinder for GitHub Issue management, /grill-with-docs for requirements, and /implement for TDD-driven execution [1].
