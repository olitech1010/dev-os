## [1.0.0] — 2026-06-13

### Added
- Universal agent system: Orchestrator, Developer, Researcher, Reviewer, Tester, DevOps, Security
- Stack support: Next.js 14, Laravel 11, Django 5, React Native (Expo)
- Universal `skills/CLAUDE.md` — base Claude Code context for all projects
- Universal `skills/cursor.md` — Cursor rules base layer
- `skills/design.md` — design principles for AI agents
- `docs/DECISIONS.md` — Architecture Decision Records template and guide
- `docs/guides/git-workflow.md` — GitHub Flow guide with conventional commits
- `docs/guides/deployment-checklist.md` — pre/post deployment checklist
- `templates/pull_request/PULL_REQUEST_TEMPLATE.md` — PR template
- GitHub issue templates: bug reports, new stack proposals
- `CONTRIBUTING.md` — full contribution guide
- `agents/AGENTS.md` — complete agent roster and workflow protocols
