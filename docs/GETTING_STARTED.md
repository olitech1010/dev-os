# Getting Started

## 1. Project Inception
Use the Architect agent or trigger the `skills/grill-me.md` protocol to brainstorm the project requirements.

## 2. Setting the Stack
Copy the relevant stack from `.agents/skills/stacks/` into your project root as a reference.
- For Next.js: `nextjs.md`
- For Laravel: `laravel.md`

## 3. Development Workflow
1. Talk to the Orchestrator.
2. The Orchestrator will delegate to the Developer.
3. The Developer will write code (using TDD) and submit it to the Reviewer.
4. The Reviewer will approve and prompt you (the Human).
5. You approve the commit.

## 4. Deployment
Once features are complete, the DevOps agent will prepare a deployment plan for human approval.
